## 三、修改优先级建议

1. **P0（严重格式错误）**：SNATCHER（缺失分隔符）
2. **P1（大量卡片共同问题）**：同一触发条件误用 `；` → 约 20 处
3. **P2（条件后标点错误）**：触发条件后用 `,` 而非 `：` → 约 16 处
4. **P3（英文标点）**：`:` `,` `;` 未改为中文标点 → 约 8 处
5. **P4（多余符号/空格）**：末尾多余 `；` 或空格 → 约 5 处
6. **P5（语义不清）**：LARGE_SCALE_DEATH、WISE_BURIAL → 建议策划重新确认文案
