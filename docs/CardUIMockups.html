设计决策说明

1 卡片比例与形态

所有风格统一使用 3.2:4.6（16:23） 竖版比例，接近标准实体卡牌尺寸（如宝可梦卡 63mm×88mm ≈ 0.716）。区别于传统 TCG 的复杂边框和大量装饰元素，采用现代数字产品的设计语言：大圆角、留白、层级阴影。

2 所属阵营表现

保留现有颜色约定（Friendly #87CEEB / Enemy #FF9F43），但通过现代方式呈现：A 风格用顶部渐变+侧边色点，B 用边框光晕，C 用 Bento cell 强调边框，D 用粗边框+硬阴影颜色，E 用层叠阴影色调+底部芯片。

3 稀有度信号

放弃传统星星/宝石，改用动态光效作为稀有度信号：Common = 无发光/灰色静态线；Uncommon = 微光边框/蓝色线条；Rare = 呼吸脉冲动画/金色发光条纹。所有风格统一使用 wallet-rarity-line 或 glass-glow 类控制。

4 交互映射

单点（Tap）：展开/折叠描述和 Status（所有风格）。长按（Long Press）：500ms 后自动展开 + Tooltip 提示 Status 名称。拖拽（Drag）：E 风格中卡片可拖拽移动，有物理倾斜反馈。所有交互仅使用这三种手势，符合项目约束。
