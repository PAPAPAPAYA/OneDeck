# 卡片 Tag 条件/收益分类报告

## 分类规则
- 依据 `docs/卡片tag规则.md` 中的 `条件映射规则`：效果描述中 `:` 之前为条件，之后为收益。
- 若一个 tag 对应的描述片段出现在 `:` 之前，归为 **条件 tag**；
- 若出现在 `:` 之后，归为 **收益 tag**；
- 若无法从描述中识别、或同时出现在条件和收益中，列为 **不确定**。
